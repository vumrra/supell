# Privacy Policy

Last updated: July 3, 2026

supell (the "extension") is a Chrome extension built for a single purpose: checking Korean spelling and grammar. This document explains what data the extension handles, where it goes, and what is stored.

## Data We Process

- **Text being checked** — When you pause while typing or select text and request a check, that text is sent to the Naver spell checker servers (`search.naver.com`, `m.search.naver.com`) for correction. The transmission happens only at the moment of the check. The developer has no access to this text and never collects or stores it anywhere.
- Content of non-text inputs such as password and number fields is excluded from checking and never transmitted.
- The full source code is publicly available on GitHub.

## Data We Store

- A single temporary token (`passportKey`) required to call the Naver API is kept in the browser's local storage (`chrome.storage.local`). It does not identify you and is refreshed automatically when it expires.
- Nothing else is stored — no checked text, no browsing history, no usage statistics.

## Data We Do Not Collect

We do not collect personally identifiable information, health information, financial information, authentication credentials, location, web history, or usage analytics. The extension contains no ads and no tracking tools.

## Third Parties

Apart from the text sent to Naver servers for spell checking, no data is ever sold or transferred to any third party. Data sent to Naver is handled under the [Naver Privacy Policy](https://privacy.naver.com/).

Questions about this policy: vvwv309@gmail.com or an issue on the GitHub repository.

# 개인정보처리방침

최종 수정일: 2026-07-03

supell(이하 "확장 프로그램")은 한글 맞춤법 검사라는 단일 목적을 위해 만들어진 크롬 확장 프로그램입니다. 이 문서는 확장 프로그램이 어떤 데이터를 다루고, 어디로 보내며, 무엇을 저장하는지 설명합니다.

## 처리하는 데이터

- **검사 대상 텍스트** — 사용자가 입력창에 작성하다 멈춘 글, 또는 마우스로 선택한 텍스트는 맞춤법 검사를 위해 네이버 맞춤법 검사기 서버(`search.naver.com`, `m.search.naver.com`)로 전송됩니다. 전송은 검사 요청 순간에만 일어나며, 개발자는 이 텍스트에 접근할 수 없고 어디에도 수집·저장하지 않습니다.
- 비밀번호·숫자 등 문장이 아닌 입력란의 내용은 검사 대상에서 제외되며 전송되지 않습니다.
- 모든 소스 코드는 GitHub에 공개되어 있습니다.

## 저장하는 데이터

- 네이버 API 호출에 필요한 임시 토큰(`passportKey`) 하나를 브라우저 내부 저장소(`chrome.storage.local`)에 보관합니다. 이 토큰은 사용자를 식별하지 않으며, 만료되면 자동으로 갱신됩니다.
- 그 외에 저장하는 데이터는 없습니다. 검사한 텍스트, 방문 기록, 사용 통계 등 어떤 것도 기록하지 않습니다.

## 수집하지 않는 데이터

개인 식별 정보, 건강 정보, 금융 정보, 인증 정보, 위치 정보, 웹 기록, 사용 통계를 일절 수집하지 않습니다. 광고·분석 도구도 사용하지 않습니다.

## 제3자 제공

검사 대상 텍스트가 맞춤법 검사 처리를 위해 네이버 서버로 전송되는 것 외에, 어떤 데이터도 제3자에게 판매·전송하지 않습니다. 네이버로 전송된 데이터의 처리는 [네이버 개인정보처리방침](https://privacy.naver.com/)을 따릅니다.
